Ce fichier a pour but d’organiser les informations d’un compte bancaire de façon structurée et compréhensible par les systèmes informatiques. Il distingue clairement les données brutes (solde, opérations, montants) de leur mise en forme, ce qui simplifie leur partage et leur exploitation entre différentes applications, comme une banque et un logiciel de comptabilité. Les images illustrant le rendu du fichier XML sont jointes en annexe.
