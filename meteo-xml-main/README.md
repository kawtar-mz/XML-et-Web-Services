# xml-xsd-xslt-svg-METEO

** Visualisation de données météo à partir de XML

Ce projet a pour objectif de transformer des données météorologiques au format XML en un graphe lisible et exploitable. L’idée est de prendre des températures de différentes villes, les structurer correctement, puis les visualiser pour mieux comprendre l’évolution de la météo.

Pensé comme un projet à la fois pédagogique et pratique, il montre comment parser un fichier XML, extraire les informations utiles et générer un graphique clair à l’aide d’outils de visualisation. Que ce soit pour apprendre, expérimenter ou poser les bases d’un projet plus ambitieux, ce repo est là pour rendre la donnée météo plus parlante que jamais.
